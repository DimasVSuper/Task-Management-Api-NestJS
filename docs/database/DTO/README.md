# DTO Documentation Index

Dokumentasi DTO berisi validasi dan penggunaan object data transfer untuk setiap modul.

## Dokumen tersedia
- `auth.md` — register dan login DTO
- `comments.md` — comment DTO
- `users.md` — create/update user DTO
- `projects.md` — create/update project DTO
- `tasks.md` — create/update/query task DTO

Gunakan DTO ini sebagai referensi validasi request di API endpoint yang sesuai.
