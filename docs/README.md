- `API.md` — dokumentasi endpoint REST API
- `NestCLI.md` — panduan menggunakan NestJS CLI untuk struktur proyek
- `MVP.md` — ruang lingkup MVP, fitur utama, dan requirement
- `database/LRS.md` — Logical Relational Schema untuk database
- `database/DTO` — dokumentasi data transfer object per modul
- `database/Entities/README.md` — dokumentasi field dan relasi entity
- `POSTMAN/README.md` — panduan Postman jika tersedia
- `POSTMAN/POSTMAN_TESTING.md` — catatan pengujian Postman

Gunakan file-file tersebut sebagai referensi untuk pengembangan, testing, dan deployment.
